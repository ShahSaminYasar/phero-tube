Ques. Discuss the scope of var, let, and const

Ans. Var = Function scoped, hoisting enabled for which it is not recommended to use 'var' due to unexpected behaviours.
Let = Block scoped, no hoisting.
Const = Block scoped; their values are constant.

Ques. Tell us the use cases of null and undefined

Ans. Null means the absence of the object i.e. the object is not defined or submitted at all. Mostly intentionally done/initialized to keep the value blank.
Undefined means the object is defined but the value is not. Occurs unintentionally.

Ques. What do you mean by REST API?

Ans. REST API uses HTTP method to perform CRUD operations which mean:
C = Create
R = Read
U = Update
D = Delete
By using these different states/types we can perform the basic operations on an API data.
